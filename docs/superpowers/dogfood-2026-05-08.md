## Header format change (intentional)

Prior pipeline emitted YAML frontmatter:

```
---
title: <title>
source: <relpath>
category: <inferred>
version: <inferred>
lang: ru
---
```

v1 emits Context7-style inline header:

```
# <Title>

Source: <relpath>
```

Per spec §4 + §16: dropped `category/version/lang` (diadok-specific, not consumed by QMD).
